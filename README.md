ClickDash

Shorten, brand, and track every click — with real‑time analytics, custom domains, and deep linking.

ClickDash is the link management platform built for creators, marketers, and the GGC universe. It's not just a link shortener — it's a complete control center for your online presence.

✨ Features

· Custom short links – clickdash.io/your-brand
· Own domains – Bring your own domain (e.g., go.yourname.com)
· Real‑time analytics – Clicks, referrers, locations, devices
· File sharing – Share any file with tracking
· Password protection – Lock links behind a password
· Deep linking – Send iOS/Android users to different destinations
· Discord login – One‑click auth via Discord
· Premium tiers – Unlock more with built‑in payments (Stripe/Cash App)

🚀 Built for the GGC Ecosystem

ClickDash is fully integrated with:

· GGC Universe – Central identity and pulse
· FanDate – Social engagement
· GPC Hub – Terminal AI interface
· PA Language – Real‑time battle reports and pulse monitoring

🛠 Tech Stack

· Backend – Node.js + Express
· Database – MongoDB (Mongoose)
· Auth – Passport.js (Discord OAuth)
· Payments – Stripe, Cash App
· Frontend – HTML/CSS/JS + Canvas game
· Hosting – Replit / Netlify

📦 Quick Start

```bash
git clone https://github.com/your-org/clickdash
cd clickdash
npm install
cp .env.example .env
# Add your Discord OAuth keys, MongoDB URI, Stripe keys
node server.js
```
